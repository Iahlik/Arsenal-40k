⚔️ Arsenal 40K

Aplicación web para gestión de listas de ejército de Warhammer 40.000, pensada para jugadores que utilizan herramientas como New Recruit o Battlescribe.

FUNCIONALIDADES

Autenticación:
- Registro de usuario
- Inicio de sesión con JWT
- Protección de rutas
- Cierre de sesión
- Recuperación de contraseña

Gestión de Listas:
- Crear listas
- Selección de facción y destacamento
- Pegado libre de listas
- Visualización y eliminación

Stack Tecnológico: MERN
Frontend: React, React Router, Context API
Backend: Node.js, Express, MongoDB, JWT, bcrypt
